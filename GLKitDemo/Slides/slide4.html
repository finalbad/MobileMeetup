Steps

  1. GLKViewController; top-level view is a GLKView

  2. "setupGL": Performs basic OpenGL setup, then sends Vertex data for objects in scene. The Vertex data includes positions & texture coordinates. Also sends the texture data using GLKTextureInfo.

  3. Implement methods:

    • - (void)update;
      Update the scene data.

    • - (void)glkView:(GLKView *)view drawInRect:(CGRect)rect;
      Draw the scene.
